# documentation/README.md

This folder contains all project documentation for the End-to-End Data Engineering Pipeline built using Python (pandas + pyodbc), SQL Server, and SSMS.

Each file provides a focused explanation of different components of the system, from architecture design to data validation, ETL logic, and instructions for running the pipeline.

Files included:
- **01_project_overview.md** — High-level explanation of the project
- **02_architecture_design.md** — Architecture, diagrams, and flow overview
- **03_data_sources.md** — Description of all data sources and schemas
- **04_etl_process.md** — Detailed ETL pipeline logic
- **05_database_design.md** — Schema design, DDL logic, fact & dimension tables
- **06_data_quality.md** — Data cleaning rules and quality framework
- **07_validation_testing.md** — Testing methods and SQL validation queries
- **08_limitations_future_work.md** — Known issues + future improvements
- **09_how_to_run.md** — Steps to run ETL pipeline locally
